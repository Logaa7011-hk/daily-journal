# 交易日志 CLI

**两阶段工作流，每日两次运行。**

## 用法

```bash
# 盘前：市场分析
python main.py morning

# 盘后：录入操作 → 自动纪律+情绪评价
python main.py review
```

## 两阶段说明

### Phase 1 — 盘前分析（`morning`）

引导依次录入：

| 模块 | 内容要求 |
|------|----------|
| 势 | 大盘情绪/指数区间/外部扰动 |
| 术 | 今日核心标的+决策框架 |
| 法 | 热点板块/资金方向 |
| 核心逻辑 | 今日押注底层理由（一句话） |
| 风险点 | 具体标的/价位/事件，必须量化 |
| 明日计划 | 每条：标的\|触发条件\|动作(仓位+止损)\|纪律预判 |

输出：`journals/YYYY-MM-DD-复盘.md`

---

### Phase 2 — 盘后复盘（`review`）

每笔操作逐条录入：

| 字段 | 说明 |
|------|------|
| 标的+方向 | 买入/卖出/加仓/减仓 |
| 触发条件 | 量化描述 |
| 仓位+止损 | 必填，否则自动判幻想预期 |
| 实际结果 | 盈亏%或金额 |

**自动纪律评判**（规则引擎）：
- ✅ **博弈概率**：有量化触发条件 + 明确止损位
- ⚠️ **幻想预期**：含模糊词（感觉/等企稳/看情况…）或无止损

**情绪评价**：冷静执行 / 恐慌割肉 / 贪婪追高 / 报复性交易 / FOMO 等

输出：追加到当日 `journals/YYYY-MM-DD-复盘.md`
